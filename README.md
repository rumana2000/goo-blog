
### goo-blog

### Technology uses
- HTML
- CSS (Responsive)
- Bootstrap
- Vanila JavaScript

### Description
In this "Goo-Blog" project, I fetched data from the "https://dummyjson.com" API and displayed the content using HTML, CSS, and JavaScript.

### functionality
- Authentication (LocalStorage)
- Load Blog Content
- Preview Blog Content
- Pagination
- Search
